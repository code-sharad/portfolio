# Video and Text Summarizer

A web application for summarizing text and video content using AI. Paste text or a video URL and get a concise summary instantly.

## Overview

Save time by getting quick summaries of long-form content. Whether it's an article, document, or YouTube video, this tool extracts the key points.

## Key Features

- **Text Summarization** – Paste any text to get a summary
- **Video Summarization** – Extract summaries from video transcripts
- **Adjustable Length** – Choose summary detail level
- **Clean UI** – Modern interface with shadcn components

## Tech Stack

- **Frontend**: React, shadcn/ui
- **AI**: LangChain, OpenAI GPT
- **Framework**: Next.js
