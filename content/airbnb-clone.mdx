# AirBnB Clone

A full-stack Airbnb clone with user authentication, property listings, and image uploads. Experience the core features of the popular vacation rental platform.

## Overview

This project replicates the essential functionality of Airbnb, including property browsing, user accounts, and booking features.

## Key Features

- **User Authentication** – Secure signup and login
- **Property Listings** – Browse and search properties
- **Image Uploads** – Add photos to property listings
- **Responsive Design** – Works on all devices
- **Map Integration** – View property locations

## Tech Stack

- **Frontend**: React, Next.js, Tailwind CSS, Framer Motion
- **Backend**: Express.js, Node.js
- **Database**: MongoDB
- **Animations**: Framer Motion for smooth transitions
