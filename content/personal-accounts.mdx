# Personal Accounts

A web app showcasing my personal social media accounts. Features a clean, modern design with links to all my profiles.

## Overview

A simple yet stylish landing page that serves as a hub for all my social media and professional profiles.

## Key Features

- **PWA Support** – Installable as an app with service worker caching
- **Responsive Design** – Looks great on all devices
- **Quick Links** – Easy access to all social profiles
- **Offline Support** – Works without internet after first visit

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Service Worker for offline support
