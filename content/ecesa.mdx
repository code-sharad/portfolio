# ECESA

A full-stack web platform for student enrollment in college workshops and events. Integrated with Razorpay for seamless payment processing.

## Overview

Built for the Electronics and Computer Engineering Student Association, this platform streamlines event registration and workshop enrollment for college students.

## Key Features

- **Event Registration** – Browse and register for upcoming workshops and events
- **Payment Integration** – Secure payments via Razorpay gateway
- **Student Dashboard** – View enrolled events and payment history
- **Admin Panel** – Manage events, participants, and payments
- **Email Notifications** – Automatic confirmation emails

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Database**: PostgreSQL
- **Payments**: Razorpay Integration
- **Deployment**: Vercel
