# CertifyPro

A bulk certificate generation platform using CSV uploads and a drag-and-drop editor. Generate hundreds of certificates in minutes.

## Overview

CertifyPro automates the tedious process of creating certificates. Upload a CSV with participant names, customize your template, and download all certificates as a ZIP file.

## Key Features

- **CSV Upload** – Bulk import participant data
- **Drag-and-Drop Editor** – Position text and elements visually
- **Font Customization** – Choose from multiple font families and sizes
- **Live Preview** – See certificate updates in real-time
- **ZIP Download** – Download all certificates at once
- **Template Saving** – Save and reuse certificate templates

## Tech Stack

- **Frontend**: Next.js, Tailwind CSS
- **Backend**: Node.js
- **Database**: MongoDB
- **Deployment**: Vercel

## How It Works

1. Upload your certificate background image
2. Design the layout using the drag-and-drop editor
3. Map CSV columns to certificate fields
4. Preview a sample certificate
5. Generate and download all certificates
